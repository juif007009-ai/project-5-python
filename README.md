Patient Health Monitoring Dataset Generation and Preprocessing

Overview

This project creates a synthetic patient health dataset and performs basic data preprocessing. The dataset contains patient information such as age, gender, blood pressure, sugar level, cholesterol, and heart rate.

Technologies Used

Python NumPy Pandas

Dataset Features

Patient_ID Age Gender Blood_Pressure Sugar_Level Cholesterol Heart_Rate

Conclusion

The project successfully generates a patient health dataset, checks data quality, handles missing values, and prepares the data for further analysis and predictive modeling.
